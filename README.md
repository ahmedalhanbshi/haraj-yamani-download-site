# Haraj Yamani Download Site

موقع تحميل رسمي مستقل لتطبيق **حراج يمني** مبني بـ Next.js وTypeScript وApp Router.

## التشغيل

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```

## التحديثات

- غيّر رابط التحميل من `src/config/app-release.ts`.
- حدّث بيانات الإصدار من الملف نفسه.
- مسار `version.json` يقرأ البيانات من نفس الملف المركزي.
